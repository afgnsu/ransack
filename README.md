##Step by Step:
1. git clone https://github.com/afgnsu/ransack.git
2. cd ransack-master
3. rvm use 2.2.0
4. rvm gemset create rails420
5. rvm use 2.2.0@rails420
6. bundle
7. vi config/database.yml
8. rake db:create db:migrate db:seed
9. rails s
